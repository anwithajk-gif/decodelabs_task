# Data Classification Using AI

## Objective
Build a machine learning model to classify Iris flower species using the K-Nearest Neighbors (KNN) algorithm.

## Technologies Used
- Python
- Scikit-learn
- VS Code

## Dataset
Iris Dataset

## Features
- Data loading and preprocessing
- Train-test split
- Feature scaling
- KNN classification
- Accuracy evaluation
- Confusion matrix generation
- Classification report generation

## Algorithm Used
K-Nearest Neighbors (KNN)

## Output
The model predicts the species of Iris flowers and evaluates performance using accuracy, confusion matrix, and classification report.
